\mbox{\hyperlink{classaderite_1_1scene_1_1_scene}{Scene}} object, at any given time a single \mbox{\hyperlink{classaderite_1_1scene_1_1_scene}{Scene}} can be active in aderite. Scenes are guaranteed to be serializable. This includes all the possible resources that it uses be it meshes, materials, etc. However these resources are loaded as trunks into the asset manager the actual data is not loaded until needed.
